## Migration steps
1. 노드가 버전이 18 + 인지 확인
2. ```corepack enable```
3. 클론 받은 폴더로 이동
4. ```yarn set version stable```
5. ```yarn -v``` 버전 확인
6. ```yarn```

## 코딩 규칙
* Page를 만들고 그안에 features 및 컴포넌트 작성
* Page 이름은 featuresPage로 통일
* 
* 커밋할 경우 앞에 [수정] or [추가] 를 기입
* localReducers 안에는 'Success', 'Failure' 로 끝나는 함수 만들지 말것
