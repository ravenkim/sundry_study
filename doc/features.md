# 바텐더 — 기능 정의서

> 1인가구를 위한 AI 바텐더 동반자 앱  
> 키워드: **외로움** (동반자·대화·루틴·연결) / **식생활** (식재료·메뉴·영양·배달 비효율) / **건강한 음주문화** (취향·여유·안전한 마무리)

## 건강한 음주문화 (전 기능 공통)

앱 전반이 “많이 마실수록 좋다”가 아니라 **한 잔의 품질·기분·기록**을 돕는 방향을 취합니다. 구현·카피·프롬프트에서 아래를 기본으로 둡니다.

| 원칙 | 제품에서의 반영 |
|------|----------------|
| **자기결정 존중** | 잔 수·도수를 경쟁하거나 수치로 밀어붙이지 않음. “오늘은 가볍게” 같은 선택을 존중 |
| **몸과 맥락** | 대화·추천 시 물·안주, 천천히 마시기, 무리한 연속 음주는 피하는 취지를 자연스럽게 제안 가능 |
| **마무리** | F-02와 동일: 과음 징후 시 부드러운 종료·휴식·물 권장. `safetyCutoffSuggested` 등 API 플래그와 연계 ([api.md](./api.md)) |
| **부적합 상황** | 미성년자·운전 예정·특정 약 복용 등 **술이 맞지 않는 맥락**이 드러나면 주류 추천을 하지 않고 무알코올·차·안주·대화로 전환 |
| **기록의 의미** | F-05 테이스팅 노트는 “들켰다” 경쟁이 아니라 **향·취향 학습·추억**용임을 톤으로 유지 |
| **투명성** | 주류 정보(도수 등)는 **정보 제공**이며, 개인 한도·법규를 대신 판단하지 않는다는 전제 유지 |

배경 서술은 [problem-definition.md](./problem-definition.md), 바텐더 말투·금지 표현은 [persona.md](./persona.md)와 맞출 것.

### F-01. 바텐더 인사 (홈)

**화면:** 홈  
**키워드:** 외로움 → 동반자

| 항목 | 내용 |
|------|------|
| 설명 | 앱 진입 시 바텐더가 시간대·날씨·최근 기록 기반으로 먼저 말을 건다 |
| 입력 | 없음 (자동) |
| 출력 | 바텐더 한 마디 + 오늘의 추천 카드 (술/안주/음악) |
| 예시 | "오늘 금요일 밤이네요. 한 주 고생하셨죠? 오늘은 버번 한 잔 어때요?" |
| API | `GET /api/greeting` |

---

### F-02. 바텐더 대화 (채팅)

**화면:** 대화  
**키워드:** 외로움 → 대화·연결

| 항목 | 내용 |
|------|------|
| 설명 | AI 바텐더와 대화. **화면 진입 시 바텐더가 먼저 말을 건다** (첫 메시지는 항상 바텐더) |
| 첫 질문 예시 | "어서오세요, 단골손님! 오늘 어떤 분위기세요? 혹시 마시고 싶은 게 있으신가요?" |
| 흐름 | 바텐더가 기분·취향을 물어보며 자연스럽게 **음료·안주·음악** 추천으로 이어짐 |
| 입력 | 텍스트 메시지 |
| 출력 | 바텐더 응답 (공감 + 추천 포함 가능) |
| 바텐더 페르소나 | 이름: 준, 말투: 친근하되 과하지 않게, 단골손님 호칭 사용 |
| 추천 방식 | 별도 화면 이동 없이 대화 중에 "오늘 날씨엔 버번 한 잔 어때요?" 식으로 자연스럽게 제안 |
| 과음 감지 | 대화 중 "많이 마셨다" 언급 시 자연스럽게 마무리 유도 |
| 건강한 음주 태도 | 잔 수 과시·“한 번에 더” 유도 금지. 취향·향·음악·안주와의 밸런스를 함께 이야기. 음주 부적합 상황이면 주류 대신 무알코올·차·간단 안주 제안 |
| API | `POST /api/chat` |

**시스템 프롬프트 핵심:**
```
당신은 혼자 사는 사람들을 위한 바텐더 '준'입니다.
손님을 '단골손님'이라고 부르고, 따뜻하고 친근하게 대화합니다.
대화 시작 시에는 반드시 먼저 기분이나 원하는 음료를 물어봅니다.
대화 흐름 속에서 자연스럽게 음료·안주·음악을 추천합니다.
과음 징후가 보이면 부드럽게 마무리하고 휴식을 권합니다.
술을 경쟁·과시의 대상으로 다루지 말고, 잔 수를 늘리도록 부추기지 않습니다.
운전 예정·미성년·약 복용 등 술이 부적절한 상황이면 주류를 권하지 말고 무알코올이나 안주·대화 위주로 응답합니다.
```

### F-03. 술·안주·음악 추천

**화면:** 오늘 뭐 마실까  
**키워드:** 식생활 → 메뉴·영양 / 외로움 → 동반자

| 항목 | 내용 |
|------|------|
| 설명 | 기분·상황 선택 → 오늘에 맞는 술·안주·음악 세트 추천 |
| 입력 | 기분 선택 (좋음/보통/우울/피곤/신남) + 선호 주종 선택 (선택사항) |
| 출력 | 술 추천 (이름·도수·특징) + 안주 1~2가지 + 음악 1곡 (YouTube 링크) |
| 건강한 음주문화 | 추천 설명에 도수·특징은 정보로만 제시하고, “천천히”“안주와 함께”“물 한 잔 곁들이기” 등 부담 없는 팁을 한두 문장 포함 가능. 피곤·우울만으로 술을 필수처럼 포장하지 않음 |
| 술 정보 | 이름, 생산지, 도수, 향·맛·피니시 설명 |
| 음악 | Claude가 어울리는 곡 선정 → `https://www.youtube.com/results?search_query={곡명}` 링크 |
| API | `POST /api/recommend` |

### F-04. 냉장고 안주

**화면:** 냉장고  
**키워드:** 식생활 → 식재료·배달 비효율 해소

| 항목 | 내용 |
|------|------|
| 설명 | 냉장고에 있는 재료 입력 → 1인분 안주 레시피 추천 |
| 입력 | 재료 태그 입력 (예: 달걀, 두부, 파, 참치캔) |
| 출력 | 레시피 이름 + 재료 + 조리 시간 + 단계별 방법 |
| 재료 없을 때 | 배달 추천 — 혼자 시키기 좋은 메뉴 + 최소주문금액 고려한 구성 제안 |
| 영양 | 레시피별 간단한 영양 포인트 (단백질/채소 균형 여부) |
| API | `POST /api/fridge/recipe` |

### F-05. 위스키 테이스팅 노트

**화면:** 테이스팅 노트  
**키워드:** 외로움 → 루틴 / 식생활 → 메뉴

위스키 기록에 최적화된 노트. 다른 주종도 기록 가능하되 위스키 전용 필드는 선택 입력으로 처리.

#### F-05-1. 노트 작성

| 항목 | 내용 |
|------|------|
| 설명 | 마신 술을 기록. 바텐더가 코멘트를 달아줌 |
| 공통 필드 (필수) | 술 이름 / 주종 / 별점 1~5 |
| 공통 필드 (선택) | 향(Nose) / 맛(Palate) / 피니시(Finish) / 한 줄 감상 |
| 위스키 전용 (선택) | 증류소 / 숙성연수 / 캐스크 종류 / 스타일·지역 (스페이사이드·아일라·버번 등) |
| 출력 | 저장 완료 + 바텐더 한 마디 코멘트 |
| 코멘트 예시 | "지난번 라프로익보다 더 마음에 드셨나봐요. 취향이 아이리시 쪽으로 가시는 것 같은데요?" |
| API | `POST /api/notes` |

#### F-05-2. 노트 목록

| 항목 | 내용 |
|------|------|
| 설명 | 그동안 기록한 술 목록을 시간 역순으로 표시 |
| 표시 항목 | 술 이름 / 날짜 / 별점 / 한 줄 감상 |
| 필터 | 주종별 필터 — 위스키(기본) / 버번 / 맥주 / 와인 / 막걸리 / 기타 |
| API | `GET /api/notes` |

#### F-05-3. 취향 분석

| 항목 | 내용 |
|------|------|
| 설명 | 위스키 기록이 5개 이상 쌓이면 바텐더가 취향 요약 제공 |
| 출력 | "단골손님은 피트향 강한 아일라 계열을 선호하시네요. 다음엔 아드벡도 도전해 보세요." |
| API | `GET /api/notes/analysis` |

### F-06. 루틴 (단골 시스템)

**화면:** 홈 + 설정  
**키워드:** 외로움 → 루틴·연결

| 항목 | 내용 |
|------|------|
| 설명 | 매일 방문하면 스트릭 쌓임. 단골 등급 상승 |
| 등급 | 첫 방문 → 3일: 단골 → 7일: 단골 VIP → 30일: 명예 단골 |
| 알림 | 저녁 시간 푸시 알림 "오늘 바 문 열었어요" |
| 주간 리포트 | 이번 주 마신 것 요약 + 바텐더 한 마디 |
| API | `GET /api/streak` / `GET /api/report/weekly` |

### TastingNote
```
id                integer  PK
drink_name        string   술 이름 (필수)
category          string   위스키/버번/맥주/와인/막걸리/기타
rating            integer  1~5 (필수)
aroma             string   향(Nose) 메모
taste             string   맛(Palate) 메모
finish            string   피니시(Finish) 메모
comment           string   한 줄 감상
-- 위스키 전용 (nullable) --
distillery        string   증류소
age_statement     string   숙성연수 (예: "12년", "NAS")
cask_type         string   캐스크 종류 (예: 셰리, 버번, 포트)
whisky_region     string   스타일·지역 (예: 스페이사이드, 아일라, 버번)
-- 공통 --
bartender_comment string   바텐더 코멘트
created_at        datetime
```
