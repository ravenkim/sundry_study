# 바텐더 — 기능 정의서

> 1인가구를 위한 AI 바텐더 동반자 앱  
> 키워드: **외로움** (동반자·대화·루틴·연결) / **식생활** (식재료·메뉴·영양·배달 비효율)

---

## 화면 목록

| # | 화면 | 경로 | 설명 |
|---|------|------|------|
| 1 | 홈 (바 카운터) | `/` | 바텐더 인사 + 오늘의 추천 카드 |
| 2 | 대화 | `/chat` | 바텐더와 자유 채팅 |
| 3 | 오늘 뭐 마실까 | `/recommend` | 술·안주·음악 추천 |
| 4 | 냉장고 | `/fridge` | 재료 입력 → 레시피·배달 추천 |
| 5 | 테이스팅 노트 | `/notes` | 음주 기록 + 히스토리 |

---

## 기능 상세

---

### F-01. 바텐더 인사 (홈)

**화면:** 홈  
**키워드:** 외로움 → 동반자

| 항목 | 내용 |
|------|------|
| 설명 | 앱 진입 시 바텐더가 시간대·날씨·최근 기록 기반으로 먼저 말을 건다 |
| 입력 | 없음 (자동) |
| 출력 | 바텐더 한 마디 + 오늘의 추천 카드 (술/안주/음악) |
| 예시 | "오늘 금요일 밤이네요. 한 주 고생하셨죠? 오늘은 버번 한 잔 어때요?" |
| API | `GET /api/greeting` |

---

### F-02. 바텐더 대화 (채팅)

**화면:** 대화  
**키워드:** 외로움 → 대화·연결

| 항목 | 내용 |
|------|------|
| 설명 | AI 바텐더와 자유롭게 대화. 기분·고민·일상을 얘기하면 공감하고 반응 |
| 입력 | 텍스트 메시지 |
| 출력 | 바텐더 응답 (공감 + 가벼운 제안) |
| 바텐더 페르소나 | 이름: 준, 말투: 친근하되 과하지 않게, 단골손님 호칭 사용 |
| 과음 감지 | 대화 중 "많이 마셨다" 언급 시 자연스럽게 마무리 유도 |
| API | `POST /api/chat` |

**시스템 프롬프트 핵심:**
```
당신은 혼자 사는 사람들을 위한 바텐더 '준'입니다.
따뜻하고 친근하게 대화하되, 과음은 부드럽게 말립니다.
손님을 '단골손님'이라고 부릅니다.
```

---

### F-03. 술·안주·음악 추천

**화면:** 오늘 뭐 마실까  
**키워드:** 식생활 → 메뉴·영양 / 외로움 → 동반자

| 항목 | 내용 |
|------|------|
| 설명 | 기분·상황 선택 → 오늘에 맞는 술·안주·음악 세트 추천 |
| 입력 | 기분 선택 (좋음/보통/우울/피곤/신남) + 선호 주종 선택 (선택사항) |
| 출력 | 술 추천 (이름·도수·특징) + 안주 1~2가지 + 음악 1곡 (YouTube 링크) |
| 술 정보 | 이름, 생산지, 도수, 향·맛·피니시 설명 |
| 음악 | Claude가 어울리는 곡 선정 → `https://www.youtube.com/results?search_query={곡명}` 링크 |
| API | `POST /api/recommend` |

---

### F-04. 냉장고 안주

**화면:** 냉장고  
**키워드:** 식생활 → 식재료·배달 비효율 해소

| 항목 | 내용 |
|------|------|
| 설명 | 냉장고에 있는 재료 입력 → 1인분 안주 레시피 추천 |
| 입력 | 재료 태그 입력 (예: 달걀, 두부, 파, 참치캔) |
| 출력 | 레시피 이름 + 재료 + 조리 시간 + 단계별 방법 |
| 재료 없을 때 | 배달 추천 — 혼자 시키기 좋은 메뉴 + 최소주문금액 고려한 구성 제안 |
| 영양 | 레시피별 간단한 영양 포인트 (단백질/채소 균형 여부) |
| API | `POST /api/fridge/recipe` |

---

### F-05. 테이스팅 노트

**화면:** 테이스팅 노트  
**키워드:** 외로움 → 루틴 / 식생활 → 메뉴

#### F-05-1. 노트 작성

| 항목 | 내용 |
|------|------|
| 설명 | 오늘 마신 술을 기록. 바텐더가 코멘트를 달아줌 |
| 입력 필드 | 술 이름 (필수) / 주종 / 별점 1~5 / 향 / 맛 / 피니시 / 한 줄 감상 |
| 출력 | 저장 완료 + 바텐더 한 마디 코멘트 |
| 코멘트 예시 | "지난번 라프로익보다 더 마음에 드셨나봐요. 취향이 아이리시 쪽으로 가시는 것 같은데요?" |
| API | `POST /api/notes` |

#### F-05-2. 노트 목록

| 항목 | 내용 |
|------|------|
| 설명 | 그동안 기록한 술 목록을 시간 역순으로 표시 |
| 표시 항목 | 술 이름 / 날짜 / 별점 / 한 줄 감상 |
| 필터 | 주종별 필터 (위스키/맥주/와인/막걸리/기타) |
| API | `GET /api/notes` |

#### F-05-3. 취향 분석

| 항목 | 내용 |
|------|------|
| 설명 | 기록이 5개 이상 쌓이면 바텐더가 취향 요약 제공 |
| 출력 | "단골손님은 피트향 강한 아이리시 계열을 선호하시네요" |
| API | `GET /api/notes/analysis` |

---

### F-06. 루틴 (단골 시스템)

**화면:** 홈 + 설정  
**키워드:** 외로움 → 루틴·연결

| 항목 | 내용 |
|------|------|
| 설명 | 매일 방문하면 스트릭 쌓임. 단골 등급 상승 |
| 등급 | 첫 방문 → 3일: 단골 → 7일: 단골 VIP → 30일: 명예 단골 |
| 알림 | 저녁 시간 푸시 알림 "오늘 바 문 열었어요" |
| 주간 리포트 | 이번 주 마신 것 요약 + 바텐더 한 마디 |
| API | `GET /api/streak` / `GET /api/report/weekly` |

## API 엔드포인트 목록

| Method | Endpoint | 설명 |
|--------|----------|------|
| GET | `/api/greeting` | 오늘의 바텐더 인사 |
| POST | `/api/chat` | 바텐더 대화 |
| POST | `/api/recommend` | 술·안주·음악 추천 |
| POST | `/api/fridge/recipe` | 재료 기반 레시피 추천 |
| GET | `/api/notes` | 테이스팅 노트 목록 |
| POST | `/api/notes` | 테이스팅 노트 작성 |
| GET | `/api/notes/:id` | 노트 상세 |
| DELETE | `/api/notes/:id` | 노트 삭제 |
| GET | `/api/notes/analysis` | 취향 분석 |
| GET | `/api/streak` | 방문 스트릭 조회 |
| GET | `/api/report/weekly` | 주간 리포트 |

## 데이터 모델

### TastingNote
```
id          integer  PK
drink_name  string   술 이름
category    string   위스키/맥주/와인/막걸리/기타
rating      integer  1~5
aroma       string   향 메모
taste       string   맛 메모
finish      string   피니시 메모
comment     string   한 줄 감상
bartender_comment string 바텐더 코멘트
created_at  datetime
```

### UserStreak
```
id            integer  PK
last_visit    date     마지막 방문일
streak_count  integer  연속 방문 일수
grade         string   첫방문/단골/단골VIP/명예단골
```
