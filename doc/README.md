## 프로젝트 개요

**앱 이름:** 바텐더 (Bartender)  
**주제:** 1인가구 해커톤 — 외로움 · 식생활  
**핵심 컨셉:** 퇴근 후 들르는 단골 바. AI 바텐더가 대화 상대이자 식생활 코치.

### 서브 키워드 매핑

| 키워드 | 해결하는 문제 | 주요 기능 |
|--------|-------------|---------|
| 외로움 | 관계 결핍, 대화 상대 없음 | 바텐더 대화, 루틴, 단골 시스템 |
| 식생활 | 1인분 식재료 낭비, 메뉴 결정 피로 | 냉장고 레시피, 술·안주 추천 |
