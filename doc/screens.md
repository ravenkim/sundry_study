# 바텐더 — 화면 정의서

> 화면·라우팅 목록과 기능 매핑의 단일 원천은 [features.md](./features.md)입니다.

---

## 디자인 시스템

### 분위기
앤틱 바(Antique Bar). 19세기 미국 살룬과 영국 펍의 감성. 어둡고 따뜻한 나무·가죽·황동 질감.

### 색상 토큰

| 토큰 | 값 | 용도 |
|------|----|------|
| `bg-primary` | `#1A0A05` | 기본 배경 (짙은 마호가니) |
| `bg-surface` | `#2C1A0E` | 카드·패널 배경 |
| `gold` | `#C9A227` | 강조·버튼 테두리·아이콘 |
| `gold-light` | `#E8C85A` | 호버·활성 상태 |
| `cream` | `#F5E6C8` | 기본 텍스트 |
| `cream-muted` | `#A89070` | 보조 텍스트·플레이스홀더 |
| `bubble-bar` | `#3D2410` | 바텐더 말풍선 배경 |
| `bubble-user` | `#1E3A2A` | 사용자 말풍선 배경 |

### 폰트
- 제목·로고: `Playfair Display` (serif, 앤틱 느낌)
- 본문: `Noto Serif KR`
- UI 레이블: `Noto Sans KR` (가독성)

### 공통 장식 요소
- 구분선: `─── ✦ ───` 또는 `⊱─────⊰`
- 버튼: 황동(gold) 테두리 + 모서리 장식 `◈ 텍스트 ◈`
- 아이콘: 선(line) 스타일, 앤틱 느낌의 SVG

## 2. 대화 (`/chat`)

**목표:** 바텐더가 먼저 말을 걸고, 대화를 통해 음료·안주·음악 추천까지 연결.  
**연결 기능:** F-02.

### 모바일 와이어프레임

```
┌─────────────────────────────┐
│ ← [홈]        대화      [📋] │  ← 상단 바

│  ┌─────────────────────┐   │
│  │ [바텐더 캐릭터 소형]  │   │  ← 바텐더 아바타 + 이름 "준"
│  │ 준                  │   │
│  └─────────────────────┘   │
│                             │
│  ╭──────────────────────╮  │
│  │ 어서오세요, 단골손님!   │  │  ← 바텐더 말풍선 (bubble-bar)
│  │ 오늘 어떤 분위기세요?  │  │

│                             │
│         ╭────────────────╮  │
```

**구성 요소**

| 요소 | 동작 |

| 말풍선 | 바텐더: 좌측 + bubble-bar / 사용자: 우측 + bubble-user |
| 추천 카드 | 대화 응답에 recommendation이 있을 때 말풍선 아래 인라인 표시 |
| 입력창 | 하단 고정, 전송 버튼 gold 색상 |
